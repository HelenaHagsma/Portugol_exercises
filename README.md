# Portugol_exercises
A repository for my projects in Portugol language.  
--> Portugol, is a programming language with syntax similar to Portuguese.  
Its main goal is to make learning programming logic easier, especially for beginners, by providing a more accessible, didactic, and intuitive approach during the first steps in software development.
